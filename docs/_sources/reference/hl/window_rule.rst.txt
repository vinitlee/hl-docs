hl.window_rule
==============

.. function:: hl.window_rule(spec)

   Register a window rule.

Signature
---------

.. code-block:: text

   hl.window_rule(spec: HL.WindowRuleSpec): HL.WindowRule

Parameters
----------

spec : :class:`HL.WindowRuleSpec`
   Rule table.

Returns
-------

rule : :class:`HL.WindowRule`
   Runtime rule object.

Examples
--------

.. code-block:: lua

   hl.window_rule({
       name = "float-dialogs",
       match = { class = "pavucontrol" },
       float = true,
       center = true,
   })

Notes
-----

Match values may be strings, booleans, or numbers. Named rules are reused on later calls.
