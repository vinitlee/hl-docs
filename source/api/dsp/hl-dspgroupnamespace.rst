hl.dspGroup
===========

Group dispatcher constructors.

.. list-table::
   :header-rows: 1
   :widths: 28 32 40

   * - Dispatcher
     - Returns
     - Description
   * - ``active(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Operate on the active group.
   * - ``lock(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Lock or unlock groups.
   * - ``lock_active(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Lock or unlock the active group.
   * - ``move_window(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Move a window within a group.
   * - ``next(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Focus the next group member.
   * - ``prev(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Focus the previous group member.
   * - ``toggle(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Toggle group state.
