   * - ``focus(...)``

   * - ``force_renderer_reload(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`

   * - ``global(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Run a global shortcut.
   * - ``layout(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`

     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`

   * - ``send_shortcut(...)``

   * - ``submap(...)``
     - :ref:`HL.Dispatcher <class-HL-Dispatcher>`
     - Switch the active submap.
